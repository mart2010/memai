history — the rationale and rejected alternatives are still authoritative — but they
are no longer maintained; the living documentation (`../AUTHORING_BUNDLES.md`,
`../PLAN.md`, root `CLAUDE.md`) supersedes them where they overlap.

(archived 2026-07-14); the living `../PLAN.md` carries only the still-open work.
